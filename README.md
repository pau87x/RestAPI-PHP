# RestAPI-PHP 
A little RESTful api with PHP

#### Requirements 
PHP >5.4  
MySQL  

#### Installation 
Clone or download this repo . 
Create database, change config in the DBConnect.php 

##### Routes
| ACTION        |  METHOD  |  URL                      | 
| ------------- |:--------:| --------------------------|
| Get users     |  GET     |  "/api/v1/users.php"      |  
| Get user      |  GET     |  "/api/v1/users.php?id=1" | 
| Create user   |  POST    |  "/api/v1/users.php"      | 
| Update user   |  PUT     |  "/api/v1/users.php"      | 
| Delete user   |  DELETE  |  "/api/v1/users.php"      |
